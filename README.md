# Guinée Web

## Project Description

Guinée Web is a social network application designed to connect users, share content, and engage with various forms of media. The platform aims to provide an immersive social experience, combining essential features of popular social media applications with unique enhancements tailored for its users.

### Key Features

- **User Profiles**: Customize personal profiles with profile pictures, bio descriptions, and interests.
- **News Feed**: A dynamic feed that displays posts from friends, trending topics, and personalized content.
- **Messaging**: Private and group messaging options to stay connected with friends and family.
- **Likes and Comments**: Interact with posts by liking and commenting, fostering community engagement.
- **Groups and Events**: Create or join groups based on interests and organize events to engage with the community.
- **Photo and Video Sharing**: Easily upload and share photos and videos, enhancing the sharing experience.
- **Notifications**: Stay updated with real-time notifications about likes, comments, messages, and friend requests.
- **Privacy Controls**: Manage who can see your posts and personal information to ensure a safe user experience.
- **Search Functionality**: Discover other users, posts, and groups through an intuitive search interface.

### Technologies Used
- **Frontend**: React.js, HTML, CSS, and JavaScript
- **Backend**: Node.js, Express.js, and MongoDB
- **Authentication**: JWT (JSON Web Tokens) for secure user authentication
- **Deployment**: Hosted on AWS for scalability and performance

### Future Enhancements
- **Integration with Other Social Media Platforms**: Allow users to link their other social media accounts.
- **Advanced Privacy Settings**: Enable more detailed controls over who can access personal data.
- **AI-Driven Recommendations**: Implement machine learning algorithms to suggest friends and content based on user behavior.

### Conclusion

Guinée Web aims to create a safe and engaging social network experience where users can interact freely and connect meaningfully. Join us in building a vibrant community focused on sharing, engaging, and celebrating life!
